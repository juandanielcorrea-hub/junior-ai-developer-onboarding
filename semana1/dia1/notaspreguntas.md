¿Qué es un LLM?

Un modelo de lenguaje entrenado para predecir el siguiente token utilizando enormes cantidades de texto.

¿Qué es un token?

La unidad básica de texto que procesa el modelo.

¿Qué es un embedding?

La representación vectorial de un token o texto.

¿Qué hace el Transformer?

Procesa relaciones entre tokens usando mecanismos de atención.

¿Qué es el pretraining?

Entrenamiento masivo para aprender patrones generales del lenguaje.

¿Qué es el fine-tuning?

Especialización del modelo para tareas concretas.

¿Qué es RLHF?

Alineación del modelo usando preferencias humanas.

¿Qué es la inferencia?

La generación de respuestas cuando el usuario envía un prompt.
